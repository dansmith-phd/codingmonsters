**Title:** *Coding Monsters: Applied R Workshops for Sport, Exercise, Health, and Social Sciences*

**Description:** *A curated repository of open-source datasets and R scripts developed for a university-wide workshop series. Designed to upskill graduate-level researchers, PhD candidates, and faculty in data hygiene, exploratory data analysis, and basic statistical modeling using R.* 

**Topics Covered:** *Intro to object-oriented programming, basic R syntax, data management, visualization, descriptive stats, inferential stats (t-test, correlation, regression).*
